## 版本更新说明

### 如何更新版本

1. **修改此文件**: 在version.md的顶部添加新版本记录
2. **更新主模板**: 修改workflow_template.md头部的版本号和更新时间
3. **保持一致性**: 确保两个文件中的版本信息保持同步

### 版本编号规则

- **主版本号**: 重大功能变更或架构调整时递增
- **次版本号**: 新增功能或重要改进时递增
- **修订版本号**: 错误修复或小幅优化时递增

### 记录格式

```markdown
- **vX.Y** (YYYY-MM-DD): 版本标题
  - **变更类型**: 具体变更内容描述
  - **变更类型**: 具体变更内容描述
```

常用变更类型：
- **新增功能**、**优化改进**、**修复错误**、**重构代码**、**更新文档**等
